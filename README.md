# Frontend Engineering

A curated collection of frontend applications built with **React**, **JavaScript**, and **Tailwind CSS**, focused on building a strong foundation in modern frontend engineering principles.

This repository showcases projects that emphasize clean architecture, reusable components, React Hooks, performance optimization, state management, browser APIs, and responsive user interface development.

---

## Objectives

- Build production-quality frontend applications
- Strengthen React fundamentals through practical implementation
- Apply modern JavaScript and component-driven architecture
- Explore browser APIs and real-world UI interactions
- Write clean, maintainable, and scalable code
- Follow engineering practices used in modern product companies

---

## Projects

| Project | Description | Key Concepts |
|---------|-------------|--------------|
| 🎨 **ColorShift** | Interactive application that dynamically changes the page background based on user-selected colors. | React, `useState`, Event Handling, Conditional Rendering, Tailwind CSS |
| 🔐 **Password Generator** | Secure password generator with configurable length, numbers, and special characters, featuring Clipboard API integration and optimized rendering. | `useState`, `useEffect`, `useCallback`, `useRef`, Clipboard API, Algorithm Design |

## Getting Started

Clone the repository

```bash
git clone https://github.com/<your-username>/frontend-engineering.git
```

Navigate to a project

```bash
cd frontend-engineering/password-generator
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
