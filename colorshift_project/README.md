# ColorShift

A lightweight React application that allows users to dynamically change the page's background color through an intuitive set of color selection buttons. This project demonstrates fundamental React concepts, including state management, event handling, and responsive UI development.

## Overview

ColorShift provides a simple interface where each button represents a different color. Selecting a button instantly updates the application's background, showcasing React's declarative rendering and state-driven UI updates.

## Features

- Dynamic background color switching
- Responsive and minimal user interface
- Fast client-side rendering
- Clean, reusable component structure
- Built with modern React practices

## Tech Stack

- React
- JavaScript (ES6+)
- Tailwind CSS
- Vite

## Getting Started

### Prerequisites

- Node.js (v18 or later)
- npm

### Installation

```bash
git clone https://github.com/<your-username>/Basic_projects.git
cd Basic_projects/Colorshift_Projects
npm install
```

### Run Locally

```bash
npm run dev
```

The application will be available at:

```
http://localhost:5173
```
